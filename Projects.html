Automation project - 2
Problem Statement : On a daily basis multiple requests are received from differnt teams to perform certain steps of action on a web application depending on the parameters of the request.


Solution : I used UIPath to read the email request, store the parameters in a local variable, go to the web application and collect the data based on the received parameters. I then use a VBA macro to store this data in an Excel tracker and the bot performs the application action upon confirmation.

Human Hours Saved : 3 hours Effiency : 100 percent
